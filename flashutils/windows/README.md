# F1C100s/200s Windows Flash tools

## Install Zadig Driver
1. Open zadig driver installer 打开zadig
2. Choose 选择 "Device"->"Create New Device"
>![](res/Create%20New%20Device.png)
3. 上面填写Allwinner FEL Device， usbID填入1F3A EFE8 。
>![](res/FEL%20Driver.png)
4. 按 "Install Driver"
5. 再选择"Device"->"Create New Device"
>![](res/Create%20New%20Device.png)
6. 上面填写Allwinner DFU Device， usbID填入1F3A 1010 。
>![](res/DFU%20Driver.png)
7. 按 Click "Install Driver"
